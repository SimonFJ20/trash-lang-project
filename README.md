# trash-lang-project

## trash-js

This is the interpreter implementation in Typescript using NodeJS and Nearley

### Usage

```
trashjs <filename> [options]
Options:
    --ast       Write AST to a file called 'ast.get.json'
    --no-eval   Dont evaluate/run the file after parsing
```
